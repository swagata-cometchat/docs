> This page indexes focused, task‑oriented feature guides for the Android UI Kit. Each guide shows how to implement a specific capability end‑to‑end using UIKit components plus Core / Calls SDK APIs.

## When to Use These Guides

Use these guides after you have completed the base [Getting Started](/ui-kit/android/getting-started) setup and understand the core layout of your chat experience. They are ideal for layering additional UX capabilities without reinventing common patterns.

## Guide Directory

| Guide | Description |
|:------|:------------|
| [Block / Unblock User](./guide-block-unblock-user.mdx) | Let users block or unblock others in 1:1 chats; hides composer and shows unblock prompt while preventing outgoing/incoming messages. |
| [Call Log Details](./guide-call-log-details.mdx) | Detailed call insights screen: metadata, participants, join/leave history, and recordings. |
| [Group Management](./guide-group-chat.mdx) | Create/join groups, view members, add / ban users, change scopes, transfer ownership. |
| [Message Privately](./guide-message-privately.mdx) | Launch a direct 1:1 chat from a profile/list; optionally send initial message to surface conversation. |
| [New Chat](./guide-new-chat.mdx) | Unified entry for starting new 1:1 or group chats with tabbed Users / Groups search + navigation. |
| [Threaded Messages](./guide-threaded-messages.mdx) | Threaded replies: open parent message context, list replies, compose with parent linkage. |
